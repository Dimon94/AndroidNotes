### 类型转换
```python
hours = raw_input 'Enter hours'
rate = raw_input 'Enter rate'
money = float(hours) * float(rate)
print money
```

### if语句
```python
if x < 2 :
  print 'Small'
elif x < 10 :
  print 'Medium'
else :
  print 'LARGE'
print 'All done'
```
